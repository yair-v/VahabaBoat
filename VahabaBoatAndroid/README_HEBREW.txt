Vahaba Boat Android PRO - Alerts Only

שינוי חשוב:
אין יותר התראות סטטוס.
אין התראות שהמערכת מחוברת.
אין התראות שההתראות כבויות.
אין התראות על תקשורת/שרת.

יש רק 3 התראות אמיתיות:
1. טמפרטורה גבוהה מהסף שהוגדר
2. לחות גבוהה מהסף שהוגדר
3. מפלס מים גבוה מהסף שהוגדר

הערה:
אנדרואיד דורש Notification שקטה עבור שירות רקע.
היא מוגדרת מינימלית ושקטה ככל האפשר.
האזהרות הקוליות/הקופצות יופיעו רק בשלושת המצבים למעלה.

התקנה:
1. הסר את האפליקציה הישנה מהטלפון.
2. פתח Android Studio.
3. Open -> בחר את התיקייה.
4. Build -> Generate App Bundles or APKs -> Generate APKs.
5. התקן את app-debug.apk.
